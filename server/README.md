# MemeTalk - Server

Server code for MemeTalk.

## Prerequisites

- Install [Java Development Kit (JDK) 11](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html).
- Install [Gradle 6.5.1](https://gradle.org/install/).

## Build

```bash
$ ./gradlew build
```

## Run

```bash
$ ./gradlew run
```

## Test

```bash
$ ./gradlew test
```
